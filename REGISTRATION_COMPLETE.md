# ✅ Форма регистрации обновлена!

## 🎉 Что сделано

Доработана форма регистрации с полными данными пользователя.

---

## 📋 Изменения

### **Регистрация (новый пользователь):**
Теперь требуются поля:
- ✅ **Телефон** (обязательно)
- ✅ **ФИО** (обязательно)
- ✅ **Email** (опционально)
- ✅ **Адрес доставки** (обязательно) ← НОВОЕ

### **Вход (существующий пользователь):**
- ✅ Только **телефон** и **SMS код**
- Никаких дополнительных полей!

---

## 🔄 Как работает

### Сценарий 1: Новый пользователь
```
1. Вводит телефон → "Получить код"
2. Система: "Вы новый пользователь"
3. Показываются поля: ФИО, Email, Адрес
4. Заполняет данные → "Зарегистрироваться"
5. Получает SMS код → Вводит → Вход!
```

### Сценарий 2: Существующий пользователь
```
1. Вводит телефон → "Получить код"
2. Получает SMS код → Вводит → Вход!
```

## 🗄️ База данных

**Миграция применена:**
```sql
ALTER TABLE users 
ADD COLUMN IF NOT EXISTS address VARCHAR(500);
```

**Модель User:**
```python
class User(Base):
    phone = Column(String(20), unique=True, nullable=False)
    full_name = Column(String(255), nullable=False)
    email = Column(String(255), nullable=True)
    address = Column(String(500), nullable=True)  # ← НОВОЕ
    is_verified = Column(Boolean, default=False)
```
