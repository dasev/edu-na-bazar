========================================
📦 КАК СОЗДАТЬ РЕПОЗИТОРИЙ НА GITHUB
========================================

ШАГ 1: ВОЙДИТЕ В GITHUB
------------------------
Откройте: https://github.com/login
Войдите в свой аккаунт (или создайте новый)


ШАГ 2: СОЗДАЙТЕ РЕПОЗИТОРИЙ
----------------------------
1. Откройте: https://github.com/new

2. Заполните форму:
   ┌─────────────────────────────────────┐
   │ Repository name *                   │
   │ edu-na-bazar                        │
   └─────────────────────────────────────┘

   ┌─────────────────────────────────────┐
   │ Description (optional)              │
   │ 🛒 Маркетплейс прямых продаж       │
   │ от фермеров без посредников         │
   └─────────────────────────────────────┘

   ○ Public  ● Private
   (выберите Public для открытого проекта)

   ❌ НЕ ставьте галочки:
   □ Add a README file
   □ Add .gitignore
   □ Choose a license

3. Нажмите зеленую кнопку:
   [Create repository]


ШАГ 3: ЗАПУСТИТЕ СКРИПТ PUSH
-----------------------------
После создания репозитория:

1. Запустите файл: push-to-github.bat
   (двойной клик)

2. Введите ваш GitHub username

3. При запросе пароля:
   - Используйте Personal Access Token
   - НЕ обычный пароль!

4. Готово! ✅


========================================
КАК ПОЛУЧИТЬ PERSONAL ACCESS TOKEN
========================================

1. Откройте: https://github.com/settings/tokens

2. Нажмите: [Generate new token] → [Generate new token (classic)]

3. Заполните:
   - Note: "edu-na-bazar local"
   - Expiration: 90 days
   - ✅ repo (поставьте галочку на всё в разделе repo)

4. Нажмите: [Generate token]

5. СКОПИРУЙТЕ ТОКЕН! (он больше не покажется)

6. При push используйте этот токен вместо пароля


========================================
АЛЬТЕРНАТИВА: GITHUB DESKTOP (ПРОЩЕ)
========================================

Если не хотите возиться с токенами:

1. Скачайте: https://desktop.github.com/

2. Установите и войдите в аккаунт

3. File → Add Local Repository
   Выберите: c:\python\edu-na-bazar

4. Publish repository

5. Готово! Всё работает автоматически
